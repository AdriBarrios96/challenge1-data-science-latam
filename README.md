# 📊 Challenge ONE Data Science - Analisis de Ventas

Este proyecto forma parte del **Challenge ONE Data Science**. El objetivo es analizar un conjunto de bases de datos correspondientes a cuatro tiendas para extraer insights valiosos sobre su rendimiento comercial, la satisfaccion del cliente y el comportamiento de ventas. De esta manera podremos ayudar al Sr. Juan a decidir que tienda de su cadena debe vender para inicial un nuevo emprendimiento.

## 📝 Descripcion del Proyecto

A través de este análisis exploratorio de datos (EDA), procesamos y unificamos los registros de ventas de cuatro sucursales para responder preguntas clave del negocio. Desde el calculo de ingresos totales hasta la relacion entre el precio de los productos y los costos de envio, este proyecto transforma datos crudos en informacion visual y procesable.

## 🛠️ Herramientas y Tecnologias Utilizadas

* **Lenguaje:** Python
* **Manipulacion de Datos:** Pandas
* **Visualizacion de Datos:** Matplotlib
* **Entorno de Desarrollo:** Google Colab / Jupyter Notebook

## 📈 Analisis Realizados

Durante el desarrollo del cuaderno, se llevaron a cabo las siguientes operaciones operativas y analiticas:

1. **Calculo de Ingresos:** Sumatoria de la facturacion total por cada tienda para identificar la sucursal más rentable.
2. **Categorias Populares:** Agrupacion y conteo de ventas para descubrir los rubros más fuertes en cada ubicacion.
3. **Satisfaccion del Cliente:** Calculo del promedio de calificaciones de los usuarios para medir la calidad del servicio.
4. **Top Productos:** Identificacion de los articulos más y menos vendidos, presentados en formato de ranking.
5. **Analisis de Logistica:** Promedio de los costos de envio asumidos por los clientes en cada tienda.
